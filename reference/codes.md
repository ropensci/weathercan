# Meaning of climate normal 'codes'

A reference dataset containing `codes` matched to their meaning. Data
downloaded using the
[`normals_dl()`](https://docs.ropensci.org/weathercan/reference/normals_dl.md)
function contains columns indicating `code`. These are presented here
for interpretation.

## Usage

``` r
codes
```

## Format

A data frame with 4 rows and 2 variables:

- code:

  Code

- meaning:

  Explanation of the code
