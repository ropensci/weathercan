# Meaning of coded 'flags'

A reference dataset containing 'flags' matched to their meaning. Data
downloaded using the
[`weather_dl()`](https://docs.ropensci.org/weathercan/reference/weather_dl.md)
function contains columns indicating 'flags' these codes are presented
here for interpretation.

## Usage

``` r
flags
```

## Format

A data frame with 16 rows and 2 variables:

- code:

  Flag code

- meaning:

  Explanation of the code
